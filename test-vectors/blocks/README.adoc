= Block Test Vectors

Golden test vectors for FormDB block format validation.

== Files

* `superblock.hex` - Valid superblock in hex format
* `superblock.txt` - Canonical text rendering
* `document_simple.hex` - Simple document block
* `document_simple.txt` - Canonical rendering
* `invalid_checksum.hex` - Block with incorrect CRC32C
* `invalid_magic.hex` - Block with wrong magic bytes

== Usage

Test vectors are used to verify:

1. Block parsing correctness
2. Canonical rendering stability
3. Error detection (invalid blocks)

== Generation

Test vectors are generated by the reference implementation and verified manually.
