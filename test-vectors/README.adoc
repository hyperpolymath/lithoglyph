= FormDB Test Vectors
:toc:

Golden bytes and golden renders for validating implementations.

== Structure

[source,text]
----
test-vectors/
├── blocks/          # Block format test vectors
│   ├── *.bin        # Golden bytes
│   └── *.txt        # Expected renders
├── journal/         # Journal entry test vectors
│   ├── *.bin        # Golden bytes
│   └── *.txt        # Expected renders
└── fql/             # FQL query test vectors
    ├── *.fql        # Query input
    └── *.expected   # Expected output
----

== Status

Test vectors will be created as specifications are finalized.
