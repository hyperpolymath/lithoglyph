= Form.Blocks + Form.Model (Forth)
:toc:

The truth core of FormDB, implemented in Forth.

== Form.Blocks

Deterministic storage layer providing:

* Fixed-size blocks with symbolic headers
* Append-only journal
* Crash recovery
* Integrity checks + repair guidance

== Form.Model

Multi-model logical layer providing:

* Document collections
* Edge collections
* Schema + constraint metadata
* Migration artefacts
